# Изменения в программе анализа теории игр

## Цель
Модифицировать программу так, чтобы она учитывала все изначальные стратегии, даже если они были убраны как доминируемые, присваивая им вероятность 0.

## Внесенные изменения

### 1. Метод `SimplifyMatrix`
- Изменена сигнатура метода с `static double[,] SimplifyMatrix(double[,] matrix, int m, int n)` на `static (double[,] simplifiedMatrix, List<int> removedRows, List<int> removedCols) SimplifyMatrix(double[,] matrix, int m, int n)`
- Теперь метод возвращает не только упрощенную матрицу, но и списки удаленных строк и столбцов (доминируемых стратегий)
- Добавлены сообщения о том, какие стратегии были удалены как доминируемые

### 2. Метод `SolveMixedStrategy`
- Изменена сигнатура метода с `static void SolveMixedStrategy(double[,] matrix)` на `static void SolveMixedStrategy(double[,] matrix, int originalRows, int originalCols, List<int> removedRows, List<int> removedCols)`
- Добавлена логика для учета всех исходных стратегий:
  - Создаются массивы вероятностей для всех исходных стратегий игроков
  - Доминируемым стратегиям присваивается вероятность 0
  - Вычисленные вероятности для оставшихся стратегий присваиваются соответствующим позициям
- Результат выводится с указанием, какие стратегии являются доминируемыми

### 3. Главный метод `Main`
- Изменен вызов `SimplifyMatrix` для получения всех возвращаемых значений
- Изменен вызов `SolveMixedStrategy` для передачи всех необходимых параметров

## Результат
Программа теперь корректно отображает вероятности для всех исходных стратегий, включая доминируемые, которым присваивается вероятность 0, что соответствует требованиям анализа.
